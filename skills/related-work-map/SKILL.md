A literature map is an asset you maintain, not a search you re-run. As a project grows, the field's structure — who did what, which methods descend from which, where claims conflict, what is unsolved — should live in one evolving map you can query, not be reconstructed from memory each time you write.

## Discipline
- **Keep one living artifact** → a single durable map (table, citation graph, or structured notes) of the area, updated as you read — not a fresh search per paper.
- **Capture contribution, not just citation** → for each key work, record in your own words what it actually adds and where it stops (→ reading-notes-discipline), not its abstract.
- **Encode relationships** → method lineages, shared datasets, and claims that agree or conflict; the edges are the value, not the node list.
- **Mark the gaps** → the map's payoff is the unanswered questions and untried combinations it makes visible — these feed `research-ideation`.
- **Keep it reusable** → structured so the same map serves a paper's related-work, a proposal's background, and onboarding a new student.

## Anti-patterns
- Re-deriving the field from scratch for every manuscript.
- A flat citation list with no relationships or stated contributions.
- A map of only the work you cite favorably, missing the conflicting results.
- Copying abstracts instead of stating each work's actual contribution.
- Letting it rot — a map not updated as you read is worse than none.

## Hard vs adaptable
- **Hard rule:** each entry records the work's actual contribution and its relationships, not just a citation, and conflicting work is included.
- **Adaptable:** the form (table, Zettelkasten, citation graph, doc) and granularity scale to the program's size.

## Related
- `prior-art-check` — the one-time novelty gate; this is the durable structure it draws on and feeds.
- `reading-notes-discipline` — per-paper intake that populates the map.
- `research-ideation` — the map's visible gaps become candidate research directions.
- `assumption-ledger` — records your project's assumptions; this records the field's structure.
